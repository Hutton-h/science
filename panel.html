一键部署

all=y sub=y bash <(curl -Ls https://raw.githubusercontent.com/Hutton-h/science/main/science.sh)

使用说明

1. 勾选需要的协议，端口留空=随机生成，标注"CDN"的协议建议填入CDN回源端口

2. vlpt / xhpt 为 Reality 系列协议，需要 reym 指定伪装域名

3. vxpt / vwpt / vmpt 走 CDN，需要 dnym + cdnym

4. Argo 隧道需要同时填写 argo 模式 + agn 域名 + agk Token

5. 点击"生成命令"后复制，在服务器上执行即可一键部署
